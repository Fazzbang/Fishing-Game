# Assets folder

Image files used by the game:

- `tileset.png` - a strip of 16x16 pixel-art ground tiles (grass, path,
  water, sand, flowers, fence), in the same order as the tile numbers used
  in `src/map.js`.
- `shack.png` - the player's fishing shack, a decorative building placed on
  the map.
- `player.png` - the player character sprite.

All original artwork made for this project (not copied from any existing
game) - simple and small so it's easy to redraw/replace as the game grows.
